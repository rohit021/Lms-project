# Steps to Run this code in local

## Step 1: install the dependencies

- for server

```bash
npm install
```

- for client side

```bash
cd client && npm install
```

## Step 2: Now run the Project by cmd

- for server

```bash
npm run server
```

- for client

```bash
npm run client
```

- for both server and client

```bash
npm run dev
```
